# Rad UI (راد)

An RTL-first, Shadcn-inspired component library for Farsi speakers that works on both Web (React) and Mobile (React Native).

## Structure

- `apps/web` - Next.js 14+ landing page and documentation
- `apps/mobile` - Expo React Native app
- `packages/ui` - Shared UI component library
- `packages/config` - Shared ESLint and TypeScript configurations

## Getting Started

```bash
# Install dependencies
npm install

# Run all apps in development
npm run dev

# Build all packages
npm run build
```

## Tech Stack

- **Monorepo**: Turborepo
- **Web**: Next.js 14+ (App Router), TypeScript, Tailwind CSS
- **Mobile**: Expo, React Native, TypeScript, NativeWind
- **UI**: Shared component library with RTL support
